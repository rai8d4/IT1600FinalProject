# What Are Git and GitHub?
Git is a version control system that allows you to track changes to files over time. It is a very popular system used in software development.
GitHub is a website that hosts Git repositories online.

## Why Do We Use Git?
- Keeps a history of changes
- Allows you to undo mistakes
- Makes collaboration easier
- Ensures good communication practices

## Git vs GitHub (Visual Guide)
| Git | GitHub |
|----|--------|
| Runs locally on your computer | Runs online |
| Tracks file changes | Stores repositories |
