# Git & GitHub for Absolute Beginners

**Name:** Rayyan Iqbal | INFOTC 1600 Final Project

## Overview
This tutorial introduces Git and GitHub to absolute beginners. It explains what version control is, how Git and GitHub work together, and walks through a basic flow for creating a repository, making changes, and pushing those changes to GitHub.

## Target Audience
This tutorial is intended for students and beginners with little to no prior experience using Git or GitHub. This tutorial is geared towards college freshmen who are not familiar with version control systems.
